A SASS & Flexbox Framework.

Zero is a personal framework which I’ve put together so I can learn front-end development at my own pace, in my own way. I’ll also be using it for future web projects so I can continue to expand and change this as learn.

source

12 Column Grid

The standard in every framework, an easy to use, 12 column grid. Simply add the class col-x (x = between 1 - 12)

<section>
  <div class="container">
    <div class="row">
      <div class="col-12"></div>
    </div>
  </div>
</section>

Type

The type is very simple, h1 starting at 46px, line-height at 52px. Dropping down by 6 for each.



I can also add some helper classes (more below) to enable font-weights or other text styles.

Helper Classes

Zero is slightly modular, I've created some helper classes to attach to elements so I can quickly create things like margins, padding or alignments.

.center

Allows content to be vertically and horizontally aligned.

.buffer

Adds a buffer to an element, either sm or lg to create spacing around it.

.text-{alignment}

Quickly align copy to either left, right or center.
